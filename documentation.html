Introduction

Welcome to the official documentation for the Dispatch360 source code, describing in detail how to edit the code. Dispatch provides a powerful system for managing PEDs. Calls will be coming soon.

Installation

  • Download the source code from the link below.
  • Ensure Python 3.x and Visual Studio Code (VSC) are installed.
  • Open cad.py in VSC and run it using the run button.

Usage

  • Create a Trello board and set your Trello and Atlassian account up properly, tutorial provided.
  • Log in with an account number and password (provided on download).
  • Navigate to the record you want to view.
  • Delete or add records using the buttons at the bottom of your screen.

Features

  • Modern, user-friendly interface with Tkinter.
  • Secure role-based access system.
  • View live PEDs.
  • Modification of PEDs can be done through the Trello board you create.
