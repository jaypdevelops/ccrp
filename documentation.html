Introduction

Welcome to the official documentation of our source code. This project provides a comprehensive system for managing and interacting with the police database, including managing records, dispatch calls, and user authentication.

Installation

Follow the steps below to get the system up and running:

  • Download the source code using the link in the 'Download' section.
  • Ensure you have Python 3.x, and Visual Studio Code (VSC) installed on your system.
  • When in VSC, press "File", "Open File", then find the cad.py that was downloaded.
  • Run the application by pressing the run button in the top right.

Usage

This system allows users to:

  • Log in using a collar number and password.
  • Search for individual records, view citations, arrests, and more.
  • View dispatch calls and details.
  • Download source code and gain access to additional features.

Features

Key features of this project include:

  • Intuitive user interface built with Tkinter.
  • Ability to search for police records by name.
  • Management of citations, vehicles, arrests, and more.
  • Real-time dispatch call logs.
  • Role-based authentication and access controls.
